<!-- Hand-computed reference for test assertions. Use these exact values. -->

Inputs: w0=0.5, g=0.1
Config: lr=0.001, beta1=0.9, beta2=0.999, eps=1e-8
Initial state: m0=0.0, v0=0.0, step=0 (will be incremented to 1 inside apply_gradients)

Step 1 (f64 precision for reference):
  m1 = 0.9 * 0.0 + (1.0 - 0.9) * 0.1 = 0.01
  v1 = 0.999 * 0.0 + (1.0 - 0.999) * 0.1^2 = 0.001 * 0.01 = 0.00001
  bias1 = 1.0 - 0.9^1 = 0.1
  bias2 = 1.0 - 0.999^1 = 0.001
  m_hat = 0.01 / 0.1 = 0.1
  v_hat = 0.00001 / 0.001 = 0.01
  update = lr * (m_hat / (sqrt(v_hat) + eps))
         = 0.001 * (0.1 / (sqrt(0.01) + 1e-8))
         = 0.001 * (0.1 / (0.1 + 1e-8))
         ≈ 0.001 * 0.9999999 ≈ 0.000999999900...
  w1 = 0.5 - 0.000999999900 ≈ 0.499000010...

Use expected: 0.499_000_01_f32 with epsilon 1e-5.
The value 0.499000010 in f32: (0.5 - 0.001) = 0.499, residual ~1e-8 rounds away.
Safe assertion: assert!((w1 - 0.499_f32).abs() < 1e-3) is too loose.
Use: assert!((w1 - 0.499_000_01_f32).abs() < 1e-5)
