<objective>
Remove the x86-v4 (AVX-512) feature from two locations where it is hardcoded into gemm dependencies. This is a safety fix: any CPU without AVX-512 support (most x86-64 hardware pre-2019 and many current consumer CPUs) will crash with SIGILL when executing code compiled with this feature. The gemm crate automatically selects the best available ISA at compile time via RUSTFLAGS target-cpu=native — the explicit x86-v4 feature is both unnecessary and dangerous.

Purpose: Prevent SIGILL crashes on non-AVX-512 CPUs without any performance regression.
Output: Two files with x86-v4 removed from gemm features. Volta and benchmark crate compile cleanly on any x86-64 CPU.
</objective>
