- Cargo.toml with gemm `x86-v4` removed

  <name>Task 1: Remove x86-v4 from Cargo.toml gemm features</name>
  <files>Cargo.toml</files>
  <action>
    Edit line 22 of Cargo.toml. Change:
    ```
    gemm = { version = "0.19", features = ["rayon", "x86-v4"] }
    ```
    to:
    ```
    gemm = { version = "0.19", features = ["rayon"] }
    ```

    That is the only change. Do not touch any other line or dependency.

    Why: `x86-v4` is a compile-time feature that hard-enables AVX-512 code paths. On AArch64 these paths do not exist. Removing it still gives AVX2 on x86_64 via the gemm crate's runtime CPU detection, and enables the crate's own ARM NEON optimizations on AArch64. This is prerequisite for the macos-aarch64 matrix entry to build in CI.
